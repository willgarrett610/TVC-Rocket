# TVC-Rocket Build Log

Hello! Here, I will be documenting the process of building a TVC model rocket! If you don't know what that means, check out [BPS.SPACE](https://www.youtube.com/channel/UCILl8ozWuxnFYXIe2svjHhg) on YouTube.

### Saturday, July 25th, 2020

Today I have a completed flight computer (Without working software)!

The parts I'm using are

- ESP8266 Board (What I had lying around. I don't really need the internet capabilities)
- W25Q64 8MByte Flash Chip
- BMP180 GY-68 Barometric Pressure Sensor
- LSM6DS33 IMU
- MP1584EN Buck Converter
